二、版头 6 要素——公文的「身份证」

要素位置什么情况下出现易错点
① 份号左上角第一行只有涉密公文才标注份号非涉密公文标份号 → 错。份号是 6 位数字
② 密级和保密期限份号下方只有涉密公文才标注密级分三级：绝密/机密/秘密。「绝秘」是错别字。保密期限写法：「机密★10年」
③ 紧急程度密级下方（无密级则右上角）只有紧急公文才标注分两级：特急/加急。「紧急」不是法定标注
④ 发文机关标志居中每份公文必有俗称「红头」。格式：全称/规范化简称 + 文件（或只写机关名称）。字体红色大字
⑤ 发文字号发文机关标志下方居中（下行文）或左空一字（上行文）每份公文必有格式：XX发〔2026〕X号。年份用六角括号〔〕，不用方括号[]或圆括号()
⑥ 签发人右上角只有上行文才标注签发人姓名下行文/平行文标签发人 → 错。联合上行文，所有联署机关负责人都要签

⚠️ 考场最高频陷阱——发文字号的括号：

发文字号必须用六角括号〔〕。判断题里出现方括号[]或圆括号()的发文字号，一律判错。例：

国办发[2026]1号 → ❌（方括号错了）

国办发(2026)1号 → ❌（圆括号错了）

国办发〔2026〕1号 → ✅

💡 版头记忆口诀：「涉密才有份号密级，紧急公文标急字；红头大字机关名，字号六角不方不圆；上行必签签发人，下平不签别多手。」

📝 自测：
1.【判断】所有公文都应当标注份号。（×——只有涉密公文才标）
2.【判断】紧急公文的紧急程度分为「特急」「加急」「紧急」三级。（×——只有特急/加急两级）
3.【判断】签发人标识出现在所有上行文、下行文、平行文的版头。（×——只有上行文）

三、主体 9 要素——公文的「核心内容」

⑦ 标题——考得最多、错得最多的要素

规则正确示例错误示例
发文机关 + 关于 + 事由 + 的 + 文种国务院关于表彰……的决定——
「关于」和「的」不可省略国务院关于推进……的意见国务院推进……意见 ❌
除法规/规章加书名号外不用标点关于学习贯彻《信访工作条例》的通知关于做好防汛、防火工作的通知 ❌（顿号多余）
分行不断词「关于」在同一行关｜于（不能断成两行）
三个以上机关联合行文可省发文机关名称关于XXX的通知（省略了多家机关）——

⚠️ 标题最致命的三个错误：
① 少了「关于」→ 扣分（单选题的判断依据往往是这个）
② 多加了标点符号 → 扣分（标题原则上不用标点，除非书名号括法规名）
③ 标题里出现了两个「关于」→ 扣分（如「XX关于转发XX关于XX的通知的通知」）

⑧ 主送机关——「谁来办」

规则说明
位置标题下方左侧，顶格，加冒号
泛行文公报/公告/通告等公开发布的公文不写主送机关
多个主送按级别高低排列，同级的按习惯顺序
普发性下行文可用泛称：「各省、自治区、直辖市人民政府，国务院各部委、各直属机构：」
请示只能写一个主送机关

⑨ 正文——「说什么」

  • 首页必须显示正文——如果标题和主送太长导致正文被挤到第二页，格式错误
  • 结构层次序数：第一层「一、」→ 第二层「（一）」→ 第三层「1.」→ 第四层「（1）」
    注意：第三层是「1.」后面加实心圆点，不是顿号
  • 引用公文：先引标题，后引发文字号。例：根据《国务院关于XX的通知》（国发〔2026〕1号）

⑩—⑮ 附件、署名、日期、印章、附注

要素核心规则
⑩ 附件说明正文下方空一行，写「附件：」后列附件名；附件名称后不加标点；附件不用书名号
⑪ 署名 + ⑫ 日期 + ⑬ 印章印章「骑年盖月」——上沿压署名、下沿压日期；成文日期用阿拉伯数字；联合行文以最后一个签发人签发日期为准
⑭ 附注公文印发传达范围等说明。请示的附注写联系人姓名和电话
⑮ 附件另面排版，在版记之前。与正文一起装订。附件是独立的材料，不是公文的组成部分

⚠️ 成文日期是超级高频考点：
① 以负责人签发日期为准（不是拟稿/核稿/印发日期）
② 联合行文：最后一个签发机关负责人签发日期（不是第一个）
③ 电报：以发出日期为准
④ 用阿拉伯数字标注（如 2026年5月30日），不用汉字小写

四、版记 3 要素——公文的「邮寄清单」

要素说明
⑯ 抄送机关除主送机关外需要执行或知晓公文内容的机关。请示未获批复前不得抄送下级
⑰ 印发机关和印发日期印发日期晚于或等于成文日期（先签后印）。印发日期用阿拉伯数字
⑱ 页码版心下方。两页以上的公文必须标注页码

📝 自测：
1.【判断】公文标题中一般不用标点符号。（√）
2.【单选】公文成文日期以：A.拟稿日期 B.核稿日期 C.签发日期 D.印发日期 → C
3.【判断】联合行文的成文日期以第一个签发人签发日期为准。（×——最后一个）
4.【判断】印发日期可以早于成文日期。（×——先签后印，印发≥成文）

五、公文拟制与办理——一份公文的一生

公文不光是写出来的，还涉及拟制、办理、管理三大流程。考场一般就两道题——一道判断题、一道单选题。

公文拟制三步骤

步骤干什么管理主体
起草拟稿、调研、征求意见、修改——形成送审稿承办部门
审核发文机关办公厅（室）进行审核——检查内容是否符合法律法规和现行政策、是否符合公文格式规范办公厅（室）
签发负责人签署意见并签名——签发后即成定稿，随后进入印制程序机关负责人

公文办理——收文和发文两条线

类型程序
收文办理签收 → 登记 → 初审 → 承办 → 传阅 → 催办 → 答复
发文办理复核 → 登记 → 印制 → 核发

💡 考场高频考点：① 签发是公文拟制的最后一个环节（签发了才算定稿，签发日期就是成文日期）
② 审核 ≠ 签发——审核是办公厅做，签发是负责人做
③ 收文的第一个环节是签收，不是登记

📝 自测：
1.【判断】公文拟制的最后一个环节是审核。（×——是签发）
2.【判断】公文签发后如需作实质性修改，应当重新履行签发程序。（√）
3.【单选】收文办理的第一个环节是：A.登记 B.初审 C.签收 D.承办 → C

下半场：马哲 · 唯物辩证法

马哲在《综合知识》思想政治板块里占大头。而唯物辩证法是马哲里占大头。把唯物辩证法拿下，马哲的客观题至少锁住一半。今天集中攻三个部分：两大特征 → 三大规律 → 五对范畴，最后加一个认识论。

一、唯物辩证法的总框架——先挂地图

层次内容角色
两大特征普遍联系 + 永恒发展唯物辩证法的「总观点」
三大规律对立统一 + 量变质变 + 否定之否定唯物辩证法的「核心」
五对范畴原因结果 / 必然偶然 / 可能现实 / 内容形式 / 本质现象唯物辩证法的「基本环节」
认识论实践与认识 / 真理 / 认识的辩证过程唯物辩证法的「应用」

整个唯物辩证法就一句话：世界是普遍联系和永恒发展的，这个发展遵循三个规律（对立统一、量变质变、否定之否定），通过五对范畴来展开。

二、两大特征——唯物辩证法的总观点

普遍联系永恒发展
核心命题一切事物都处在相互联系之中，没有孤立存在的事物一切事物都处在运动、变化、发展之中
考场对应成语唇亡齿寒、城门失火殃及池鱼、牵一发而动全身士别三日刮目相看、长江后浪推前浪、与时俱进

📝 自测：【单选】「城门失火，殃及池鱼」体现的哲学原理是：A.量变引起质变 B.事物是普遍联系的 C.内因是变化的根据 D.新事物必然战胜旧事物 → B

三、三大规律——唯物辩证法的核心

规律一：对立统一规律（矛盾规律）——唯物辩证法的实质与核心

1. 矛盾 = 对立统一

矛盾的两种基本属性：同一性（相互依存、相互转化）和斗争性（相互排斥、相互对立）。两者同时存在，不可分割。

2. 矛盾的普遍性与特殊性——共性与个性的关系

普遍性特殊性
含义事事有矛盾、时时有矛盾每个事物的矛盾各有特点
关系普遍性寓于特殊性之中，并通过特殊性表现出来。两者是共性与个性的关系，不是整体与部分的关系

考场最大陷阱：「矛盾的特殊性寓于普遍性之中」→ ❌错误！是普遍性寓于特殊性之中。记反就翻车。

3. 主要矛盾与次要矛盾，矛盾的主要方面与次要方面

主要矛盾 vs 次要矛盾矛盾的主要方面 vs 次要方面
适用场景复杂事物中多个矛盾之间的关系一个矛盾内部的两个方面
方法论抓重点、抓关键、抓中心工作看主流、看本质、看大方向
成语匹配牵牛要牵牛鼻子、好钢用在刀刃上金无足赤人无完人、瑕不掩瑜

4. 内因与外因

内因是事物变化的根据（第一位），外因是事物变化的条件（第二位），外因通过内因起作用。考场陷阱：「外因在一定条件下起决定作用」→ ❌错误！外因永远不决定，只有内因决定。

矛盾定律口诀：矛盾 = 对立统一；普遍性寓于特殊性中；主矛抓重点、主方看主流；内因根据外因条件

规律二：量变质变规律——发展的两种状态

概念定义考场关键词
量变事物数量上的增减和构成要素排列组合的变化渐进、积累、不显著
质变事物根本性质的变化飞跃、突破、显著改变
度保持事物质的数量界限临界点、分寸、火候

三句话记住这个规律：

  1. 量变是质变的必要准备（没有量变就没有质变）
  2. 质变是量变的必然结果（量变到一定程度必然引起质变）
  3. 质变后新的量变开始（质变→新事物→新量变→新质变→循环往复）

成语匹配：不积跬步无以至千里（量变→质变）；水滴石穿（量变→质变）；过犹不及（超过「度」）

规律三：否定之否定规律——发展的方向和道路

阶段含义关键词
肯定事物保持其存在的方面现状
否定促使事物灭亡的方面——但不是全盘否定，是扬弃变革、抛弃旧形式
否定之否定在新阶段「回复」到起点的某些特征，但是更高层次的回复螺旋式上升

核心结论：事物发展的总趋势是前进的、上升的（不可逆），但道路是曲折的、迂回的（波浪式前进、螺旋式上升）。「直线前进」是错误选项。

三大规律：成语俗语→原理匹配全集

成语/俗语体现的哲学原理规律归属
牵牛要牵牛鼻子抓主要矛盾对立统一
好钢用在刀刃上抓主要矛盾
一把钥匙开一把锁矛盾特殊性（具体问题具体分析）
因材施教 / 对症下药 / 因地制宜矛盾特殊性
金无足赤，人无完人矛盾主次方面（看主流、看本质）
塞翁失马，焉知非福矛盾同一性（矛盾双方相互转化）
内因决定论 / 自强不息内因是事物变化的根据
不积跬步，无以至千里量变是质变的必要准备量变质变
水滴石穿 / 绳锯木断量变引起质变
千里之堤，溃于蚁穴量变引起质变（防微杜渐）
过犹不及超过「度」，量变引发质变走向反面
适可而止 / 掌握火候坚持适度原则
欲速则不达量变不够，质变不会发生（条件不成熟）
前途是光明的，道路是曲折的螺旋式上升、波浪式前进否定之否定
长江后浪推前浪 / 推陈出新新事物必然战胜旧事物
取其精华，去其糟粕辩证否定——扬弃
青出于蓝而胜于蓝辩证否定——在扬弃中实现超越
成语快判法：一个成语里包含两个对立面（好/坏、多/少、新/旧、快/慢）→ 对立统一；成语里有「渐」「积」「累」「滴水」→ 量变质变；成语里有「螺旋」「波折」「扬弃」「推陈出新」→ 否定之否定

辩证法与形而上学的对立——一道不会缺席的判断题

维度辩证法形而上学
世界状态联系的、发展的孤立的、静止的
发展动力事物内部矛盾（内因）外部力量的推动（外因论）
发展路径质变、飞跃（承认飞跃）只有量变、没有质变（否认飞跃）
根本分歧是否承认事物内部的矛盾性（是否承认对立统一规律）

核心结论：唯物辩证法和形而上学的根本分歧在于是否承认事物内部的矛盾性（即是否承认对立统一规律）。这道题出在判断/单选题里几乎每次都考。

📝 自测：
1.【判断】矛盾的特殊性寓于普遍性之中。（×——普遍性寓于特殊性之中）
2.【单选】「牵牛要牵牛鼻子」体现：A.矛盾特殊性 B.抓主要矛盾 C.内因决定论 D.量变质变 → B
3.【判断】外因在一定条件下可以起决定作用。（×——外因永远不起决定作用）
4.【单选】「不积跬步，无以至千里」体现：A.对立统一 B.量变引起质变 C.否定之否定 D.内因决定论 → B
5.【单选】事物发展的总趋势是：A.直线前进 B.循环往复 C.螺旋式上升 D.原地踏步 → C
6.【判断】辩证法和形而上学的根本分歧在于是否承认事物是普遍联系的。（×——根本分歧在于是否承认事物内部的矛盾性）
7.【单选】「过犹不及」体现的哲学原理是：A.抓主要矛盾 B.量变引发质变，要坚持适度原则 C.新事物取代旧事物 D.内因决定论 → B

四、五对范畴——唯物辩证法的基本环节

五对范畴在考场上一般以单选题出现，给你一个具体例子，问「体现了哪对范畴」。记住每一对的核心意思 + 一个例子就够了。

范畴核心意思经典示例
原因与结果前后相继、彼此制约。有因必有果多行不义必自毙
必然与偶然必然=规律性的确定趋势；偶然=不确定的个别现象。必然通过偶然表现时势造英雄（必然）vs 英雄个人特质（偶然）
可能与现实可能=潜在的、尚未实现的趋势；现实=已经实现的。可能变为现实需要条件梦想需要努力才能变为现实
内容与形式内容=内在要素的总和；形式=内容的结构和组织方式。内容决定形式，形式反作用于内容一部电影的剧情（内容）vs 拍摄手法（形式）
本质与现象本质=内在的、稳定的联系；现象=外在的、多变的表现。通过现象看本质透过现象看本质

💡 五对范畴考场策略：出题人最爱考的是必然与偶然和本质与现象。记住两句：「必然通过偶然为自己开辟道路」「假象也是本质的一种表现（歪曲的表现）」。这两句是判断/单选题的高频正确选项。

五、认识论——实践与认识的辩证关系

命题含义考场正确表述
实践决定认识实践是认识的来源、动力、目的、检验标准实践是检验真理的唯一标准；实践出真知
认识反作用于实践正确的认识指导实践，错误的认识误导实践理论指导实践；科学理论对实践有指导作用
认识的辩证过程感性认识 → 理性认识 → 实践 → 再认识 → 再实践认识过程的两次飞跃：①感性→理性 ②理性→实践。第二次飞跃更重要
真理真理是客观的、绝对的、相对的真理只有一个（不存在多元真理）；真理是具体的、有条件的

📝 自测：
1.【判断】实践是检验真理的唯一标准。（√）
2.【单选】认识过程的第一次飞跃是：A.实践→感性认识 B.感性认识→理性认识 C.理性认识→实践 D.理性认识→再认识 → B
3.【判断】「真理是多元的，不同的人有不同的真理」。（×——真理只有一个，是客观的）

六、唯物史观——社会历史的底层逻辑

马哲不光讲自然界和思维的规律，还讲社会历史的规律。唯物史观在考场上出现的频率略低于辩证法，但每年至少 2-3 道。

社会存在与社会意识

概念含义关系
社会存在社会生活的物质方面——地理环境、人口因素、物质资料生产方式（最重要）社会存在决定社会意识，社会意识反作用于社会存在。这是唯物史观最基本的原理
社会意识社会生活的精神方面——政治法律思想、道德、艺术、宗教、哲学等

生产力与生产关系——社会发展的最终决定力量

概念含义考场关键词
生产力人类改造自然的能力——劳动者 + 劳动资料（生产工具是标志）+ 劳动对象生产工具是生产力发展水平的客观标志；科学技术是第一生产力
生产关系人们在生产中形成的社会关系——生产资料所有制（核心）+ 人的地位关系 + 产品分配方式生产资料所有制是生产关系的基础和核心

核心规律：生产力决定生产关系（有什么样的生产力就有什么样的生产关系）；生产关系对生产力有反作用（适应就促进，不适应就阻碍）。生产力是社会发展的最终决定力量。

经济基础与上层建筑

概念含义关系
经济基础占统治地位的生产关系各方面的总和经济基础决定上层建筑；上层建筑反作用于经济基础
上层建筑政治上层建筑（政权/法制/军队）+ 观念上层建筑（意识形态）

社会基本矛盾——社会发展的根本动力

生产力 ↔ 生产关系的矛盾 + 经济基础 ↔ 上层建筑的矛盾 = 社会基本矛盾。这两对矛盾是社会发展的根本动力。改革就是通过调整生产关系以适应生产力、调整上层建筑以适应经济基础。

📝 自测：
1.【判断】社会存在决定社会意识，社会意识反作用于社会存在。（√）
2.【单选】生产力的客观标志是：A.劳动者 B.生产工具 C.劳动对象 D.科学技术 → B
3.【单选】生产关系的核心是：A.人的地位关系 B.产品分配方式 C.生产资料所有制 D.生产工具 → C
4.【判断】社会发展的最终决定力量是生产关系。（×——是生产力）

应用文练笔：写一封函

题目情景

达州市中贸粮油有限公司计划于 2026 年 6 月进行一批储备粮轮换，需要向达州市农业发展银行申请粮食收购专项贷款。请以公司名义拟一封致达州市农业发展银行的函，商洽贷款事宜。

函的格式拆解（和通知完全不同）

要素函的写法通知的写法（对比）
标题XX关于XXX的函XX关于XXX的通知
主送单一、点对点：达州市农业发展银行：可多点：各仓储管理组、财务科：
开头商洽函的开头——说明来意，简明扼要。常用「现就XXX事宜函告如下」或「现函请贵行……」会议通知开头——会议目的
正文语气平等、客气、商洽——「拟」「商请」「函请」「敬请函复」指令、告知——「现将有关事项通知如下」
结尾「特此函告，请函复」「敬请函复为盼」
不写「此致敬礼」		「特此通知」
落款发文机关署名 + 成文日期 + 公章同

⚠️ 函的致命错误清单：
① 结尾写了「此致敬礼」→ 函不是私人信件，这是硬伤
② 用了上行文的语气（「妥否，请批示」）→ 函是平行文种，不管对方级别，一律平等语气
③ 标题写成「关于XXX的请示」→ 银行和企业互不隶属，只能用函

参考范文

达州市中贸粮油有限公司
关于商请办理粮食收购专项贷款的函

达州市农业发展银行：

为做好2026年度地方储备粮轮换工作，保障区域粮食安全，我公司拟于2026年6月启动一批储备粮轮换收购，涉及稻谷约5000吨，小麦约3000吨，预计收购资金需求约3200万元。

根据国家粮食收购信贷政策及贵行相关规定，现函请贵行按照粮食收购专项贷款政策，给予我公司信贷支持。我公司将严格按规定使用贷款资金，确保专款专用，并按期还本付息。

随函附上《2026年储备粮轮换计划》及《贷款资金需求测算表》，供贵行审核参考。敬请函复为盼。

达州市中贸粮油有限公司

2026年5月30日

本篇评分要点

维度得分关键本篇表现
标题格式发文机关+关于+事由+的+函。注意「商请」体现商洽意图✅
主送点对点，银行全称准确✅
正文语气平等商洽——「拟」「函请」「敬请函复」，无指令口气✅
事实依据收购品种、数量、金额具体可查✅（稻谷5000吨/小麦3000吨/3200万元）
政策依据引用「国家粮食收购信贷政策」作为请求的依据✅
附件正文中提及附件，文后附上✅
结尾「敬请函复为盼」——平行文标准结尾。不写「此致敬礼」✅

类型用途标题关键词例
商洽函不相隶属机关之间商洽工作商请 / 拟关于商请联合开展XX工作的函
询问函向对方询问有关问题询问 / 了解关于询问XX政策适用范围的函

达州市农业发展银行

2026年6月2日

💡 来函 vs 复函的关键区别：
① 复函标题文种写「复函」（不是「函」）
② 开头第一句必须引用对方来函的标题（表示「我收到了你的函」）
③ 复函开头常见句式：「贵公司《关于XXX的函》收悉。经……，现函复如下：」
④ 复函结尾：用「特此函复」，不用「敬请函复」
