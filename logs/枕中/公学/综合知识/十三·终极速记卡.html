考前倒数第四天。今天不展开任何新内容——只做一件事：压缩。把 12 天的知识压缩成 7 张卡片，每张卡片 = 一个模块最核心的内容。考前一小时拿出来刷一遍，每刷一张就是快速激活一个模块的全部记忆。不要精读，扫读。眼睛扫到关键词，脑子里弹出相关知识点——就是这种感觉。

7 张卡片：公文 | 政治 | 宪法+行政法 | 民法+刑法 | 经济管理 | 省情科技 | 时政 | 明日：Day 14 应试策略

卡片一 公文写作

板块核心速记
15 文种口诀决必会、定可独、命最高 / 公报正式公告大、通告贴墙通知发下级 / 报告不请示、请示一文一事 / 函给平级、纪要(不叫会议纪要) / 意见多向
公告 vs 通告公告=国内外大事 / 通告=小范围+约束力。贴门上的=通告
请示 vs 报告请示=事前+需批复+一文一事+妥否请批复。报告=事后+不夹带请示+特此报告
发文字号六角括号〔〕→ 方[]圆()都错
成文日期签发日期(阿拉伯数字)。联合=最后签发人。电报=发出日期
签发人只在上行文标注。下行/平行=不标
标题公式发文机关+关于+事由+的+文种。除法规书名号外不用标点
结构层次一、→(一)→1.→(1) ←第三层是1.不是1、
附件不加书名号、不加标点
请示规则一文一事一主送、未批复不抄送下级、附注写联系人电话
四个结尾请示=妥否请批复 | 报告=特此报告 | 通知=特此通知 | 函=敬请函复为盼
版头/主体/版记版头=份号密级紧急+标志+字号+签发人(6)。主体=标题→主送→正文→附件→署名→日期→印章→附注(9)。版记=抄送+印发(3)
改错四步法①看标题(文种+格式)→②看主送(请示只能一个)→③看正文(序号+报告夹请示)→④看落款(日期阿拉伯数字+印章骑年盖月)

卡片二 政治理论

板块核心速记
马哲总框架两大特征(联系+发展)→三大规律(对立统一/量变质变/否定之否定)→五对范畴→认识论→唯物史观
对立统一矛盾=对立统一。普遍性寓于特殊性(不是反过来)。主矛=抓重点、主方=看主流。内因根据、外因条件、外因通过内因起作用。外因永远不决定
量变质变量变→质变→新量变。度=临界点。螺旋上升、不是直线
辩证法 vs 形而上学根本分歧=是否承认事物内部的矛盾性(不是「是否承认普遍联系」)
认识论实践=来源/动力/目的/检验标准。两次飞跃：感性→理性(第一次)→实践(第二次、更重要)。真理=客观、绝对、相对，一元不多元
唯物史观社会存在决定社会意识。生产力决定生产关系。经济基础决定上层建筑。生产力=最终决定力量
毛思想活的灵魂=实事求是+群众路线+独立自主。三大法宝≠灵魂(是统一战线+武装斗争+党建)
邓理论本质=解放发展生产力+消灭剥削+消除两极分化+最终共富。三个有利于。计划和市场不是本质区别
新思想·两个「最」最本质特征=党的领导(不是人民当家作主)。最大优势=党的领导
新思想·社会矛盾美好生活需要 vs 不平衡不充分(旧=物质文化vs落后生产，1981-2017，十九大改了)
新思想·首要任务高质量发展。中国式现代化=五个特征(人口巨大/共富/双文明协调/人与自然和谐/和平发展)。不是同步富裕
新发展理念创新(第一动力)、协调、绿色、开放、共享。格局=双循环不是单循环
四字诀系列四个全面=现代化国家+深改+法治+治党(旧=小康已更新)。四个意识=政治/大局/核心/看齐。四个自信=道路/理论/制度/文化。两个维护=领导核心+党中央权威。两个确立=核心地位+指导思想
三个代表+科学发展观三个代表=生产力+文化+人民利益。科学发展观=发展(第一要义)+以人为本(核心)+全面协调可持续(要求)+统筹兼顾(方法)

卡片三 宪法 + 行政法

板块核心速记
国体 vs 政体国体=人民民主专政 / 政体=人民代表大会制度(根本政治制度)
修宪全国人大修改(2/3通过) / 常委会解释。提案=常委会或1/5人大代表。公布=人大主席团(不是国家主席)
法院 vs 检察院法院=监督关系 / 检察院=领导关系。正好反着来
公民权利选举权=18岁+未被剥夺政治权利。罢工自由不在现行宪法。受教育=既是权利也是义务
复议机关部门→府或上(二选一)。府→上一府。省部→自己。派出→爹或爹的爹。垂直→只找上级
复议 vs 诉讼复议申请=60日 / 诉讼起诉=6个月 / 复议后诉讼=15日 ←三个数字互混是考场标准操作
处罚设定权法律=所有(含行政拘留) / 法规=除人身自由外 / 规章=警告+通报+一定罚款。部门规章=不能设行政许可
简易程序公民200元以下/单位3000元以下
强制措施 vs 执行措施=临时(查封/扣押/冻结) / 执行=迫使履行决定(强拆/划拨/拍卖)
复议不停止执行原则=复议期间不停止执行，例外才停止
行政诉讼被告复议维持→原机关+复议机关共被告。复议改变→复议机关。复议不作为→选原或复议
处罚/处分/许可处罚=对外(公民/组织)。处分=对内(公务员)=警告/记过/降级/撤职/开除。许可=部门规章+市级规章不能设
国家赔偿赔偿金为主。赔偿后可向故意/重大过失工作人员追偿。请求时效=2年
其他数字处罚追诉时效=2年。治安追诉=6个月。查封期限=30日(可延30)。行政拘留=一般15日↓/合并≤20日

卡片四 民法 + 刑法

板块核心速记
民事行为能力无=不满8岁(不是10岁!)。限制=8-18。完全=18或16+劳动。靠「自食其力」的16岁=视为完全
诉讼时效普通3年(知道日起)。最长20年(受损日起)
合同效力有效/无效(违法+恶意串通)/可撤销(欺诈/胁迫/重大误解/显失公平)/效力待定(限人超出+无权代理)。胁迫=可撤销≠无效
宣告失踪/死亡失踪=2年→代管财产 / 死亡=4年→婚姻消灭+继承(意外2年)。配偶再婚→不自动复婚
结婚/离婚男22女20。冷静期=30日。公证遗嘱≠最高效力→以最后遗嘱为准(民法典改了)
继承第一顺序=配偶/子女/父母。第二=兄弟姐妹/祖父母/外祖父母。有第一=无第二
侵权归责过错(一般)/过错推定(建筑物坠落)/无过错(产品/污染/高度危险/动物)/公平(双方无错)
刑责年龄不满12=无 / 12-14=特罪+核准 / 14-16=8罪(杀/重伤/强/抢/毒/爆/放/投) / 16=全责 / 18↓=从轻+无死刑
未遂 vs 中止未遂=意志以外原因(想干干不成) / 中止=自动放弃(能干不干了)。中止(无损害)=应当免除
正当防卫针对不法侵害人本人。特殊防卫(无限)=行凶/杀人/抢劫/强奸/绑架→造成伤亡不负刑责
自首 / 累犯 / 缓刑自首=「可以」从轻(不是应当)。累犯=不缓刑不假释。缓刑上限=3年有期徒刑。一般累犯间隔=5年内
贪污 vs 受贿贪污=直接拿国家钱。受贿=收别人送的钱(为他人谋利)。挪用=暂借打算还。滥用职权=故意/玩忽职守=过失。职务侵占=非国家工作人员
表见代理 / 高空抛物表见代理=外表像有代理权+第三人善意→有效。高空抛物=侵权人赔+难确定→可能加害者补偿+物业未安保→补充责任
假释 / 共同犯罪假释=有期1/2以上+无期13年以上。累犯+暴力10年以上=不假释。共犯=主犯/从犯(应从轻)/胁从犯(应减或免)/教唆犯(教唆18↓从重)
处罚三档预备=可从减免。未遂=可从减(不能免)。中止无损害=应当免。中止有损害=应当减。「可以」vs「应当」一字千金

卡片五 经济管理

板块核心速记
GDP vs GNPGDP=国土原则(境内) / GNP=国民原则(哪国人)
财政 vs 货币财政=政府(税收/支出/国债)。货币=央行(利率/准备金率/公开市场)。利率=货币≠财政
市场四种完全竞争(农产品) / 垄断竞争(餐饮) / 寡头(电信) / 完全垄断(电网)
基尼 vs 恩格尔基尼=收入差距(0.4警戒) / 恩格尔=生活水平(越小越富)
通胀 vs 通缩通胀=物价涨货币贬→紧缩政策。通缩=物价跌→扩张政策
外部性正=好处未获补偿(疫苗)。负=损害未担成本(排污)。公共物品=非竞争+非排他→政府供给
三人三理论泰勒=科学管理。法约尔=五项职能。韦伯=官僚制。西蒙=有限理性满意化(不是完全理性)
马斯洛五层：生理→安全→社交→尊重→自我实现(最高层)
PDCA / 双因素PDCA=Plan→Do→Check→Act。赫茨伯格=保健因素(消除不满)+激励因素(激发动力)。涨工资=保健≠激励
M0/M1/M2 / CPI vs PPIM0=现金 / M1=M0+活期 / M2=M1+定期+储蓄。CPI=消费端价格(通胀晴雨表) / PPI=生产端价格。机会成本=放弃的最高价值替代项
