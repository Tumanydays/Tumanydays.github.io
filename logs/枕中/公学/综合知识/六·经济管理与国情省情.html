上半场：经济常识

一、宏观经济核心概念——GDP / CPI / 失业 / 财政 / 货币

GDP——国内生产总值

考点内容
定义一定时期内一国境内生产的全部最终产品和服务的市场价值
三个关键限定①「一国境内」——不管生产者是本国人还是外国人，只要在本国国土上就计入。这和 GNP（国民生产总值）不同，GNP 按国民算，不管在哪里生产
②「最终产品」——中间产品不重复计算（如用于生产面包的面粉）。③「市场价值」——非市场交易不计入（家务劳动不计入）
名义 GDP vs 实际 GDP名义=当年价格算；实际=剔除价格变动的「真实」增长（用不变价格算）

CPI——居民消费价格指数（通货膨胀的晴雨表）

考点内容
定义反映居民家庭购买的消费品和服务价格水平变动的指标
作用衡量通货膨胀。CPI 上涨 → 购买力下降 → 通货膨胀；CPI 下降 → 通货紧缩
局限性只反映消费品价格，不反映投资品（房产/股票）价格变动

通货膨胀与通货紧缩

通货膨胀通货紧缩
表现货币贬值、物价持续上涨货币升值、物价持续下跌
影响储蓄缩水，债权人受损、债务人受益企业利润下降、失业增加
对策紧缩货币政策（提高利率/提高准备金率）扩张货币政策（降低利率/降准）

财政政策 vs 货币政策——宏观管理的两只手

财政政策货币政策
制定主体政府（财政部/国务院）中央银行（中国人民银行）
核心工具①税收（提高/降低）②政府支出（增加/减少）③国债①利率 ②存款准备金率 ③公开市场操作 ④再贴现率
扩张（刺激经济）减税 + 增加政府支出降息 + 降准 + 买入国债（放水）
紧缩（遏制过热）加税 + 减少政府支出加息 + 提准 + 卖出国债（收水）

⚠️ 考场混淆点：①「存款准备金率」是货币政策工具，不是财政政策。②扩张财政 + 扩张货币 = 「双宽松」。③财政政策工具中不包括利率——利率归央行管。

其他宏观经济指标速览

指标含义考场关键词
基尼系数衡量收入分配差距。0-1 之间，越大越不平等0.4 是国际警戒线
恩格尔系数食品支出占家庭消费总支出的比重。越高→越穷衡量生活水平的指标
失业率失业人口/劳动力人口三种失业：摩擦性/结构性/周期性
PPI工业生产者出厂价格指数反映生产端价格变动
外汇储备一国持有的外汇资产用于国际支付和稳定汇率

二、微观经济核心概念——供需 / 市场类型 / 弹性

供给与需求——市场价格的基础

概念规律影响价格方向
需求价格上升→需求量下降（需求定律）供需同时增加→价格不定；需求增+供给不变→价格涨；供给增+需求不变→价格跌
供给价格上升→供给量增加
均衡价格供给=需求时的价格。政府干预（最高限价/最低限价）会打破均衡

四种市场类型——每场考试至少一道

类型厂商数量产品差异进入壁垒典型例子
完全竞争极多无差异（同质）无农产品市场（小麦/大米）
垄断竞争很多有差异（品牌/质量）低餐饮/服装/日用消费品
寡头垄断少数几家可能有差异高电信/石油/航空
完全垄断一家无替代品极高自来水/电网（自然垄断）

💡 市场类型辨析口诀：「完全竞争无差异，垄断竞争有品牌，寡头几家相互盯，完全垄断只一家。」看到题干描述中有「少数几家大企业、相互紧盯价格决策」→ 寡头。

需求弹性——一个补充概念

类型含义典型商品
富有弹性价格变动一点→需求量变动很多奢侈品、可选消费品（豪车/旅游）
缺乏弹性价格变动很多→需求量变动很少必需品（粮食/食盐/药品）

M0 / M1 / M2——货币的三个层次

层次包含内容通俗理解
M0流通中的现金（纸币+硬币）你口袋里的钱
M1（狭义货币）M0 + 活期存款口袋里的 + 随时可取的
M2（广义货币）M1 + 定期存款 + 储蓄存款 + 其他存款社会上几乎所有「钱」

外部性与公共物品——市场失灵

概念含义举例
正外部性行为给他人带来好处，自己未获补偿打疫苗保护周边人群；养蜂授粉
负外部性行为给他人带来损害，自己未承担成本工厂排污；开车堵路
公共物品非竞争+非排他——市场不愿提供，需政府供给国防、灯塔、路灯

机会成本与边际效用递减

概念核心
机会成本做一件事而放弃的其他选择中价值最高的那个。例：用周末刷题 → 放弃了休息或社交——放弃项中价值最高的就是刷题的机会成本
边际效用递减每多消费一单位，增加的满足感越来越少。第一杯水解渴，第五杯水不想喝

三、管理常识

行政职能与公共管理

考点核心内容
行政职能计划→组织→领导→协调→控制。这是行政管理的基本职能链条
「计划」排在第一位——凡事预则立
政府经济职能宏观调控、市场监管、公共服务、社会管理、环境保护
公共管理以政府为核心、以公共利益为目标、以公共权力为基础的管理活动。新公共管理运动强调效率、市场化、顾客导向
组织结构的类型直线制（最简单）、职能制、直线-职能制（最常见）、矩阵制（项目型）、事业部制

决策理论——赫伯特·西蒙

西蒙（Herbert Simon）提出了「有限理性」决策模型。他认为管理者不是「完全理性」的——不可能掌握所有信息、比较所有方案。所以实际决策是「满意化」而非「最优化」。这是管理学核心考点。

管理学三大古典理论

理论代表人物核心主张
科学管理泰勒（Taylor）用科学方法研究工人的操作，找出「最佳工作方式」，提高效率。核心：标准化、计件工资、工时研究
一般管理法约尔（Fayol）提出管理的五项职能（计划/组织/指挥/协调/控制）和14 条管理原则。侧重管理者的视角，而非工人
官僚制韦伯（Weber）理想的组织形式——层级分明、分工明确、按章办事、非人格化。不是贬义的「官僚」，是中性的组织形式

激励理论——马斯洛与赫茨伯格

理论核心内容
马斯洛需求层次五个层次从低到高：生理 → 安全 → 社交 → 尊重 → 自我实现。低层次满足后才追求高层次。管理启示：先涨工资（生理），再给荣誉（尊重）
赫茨伯格双因素保健因素（工资/工作环境/人际）——不满足就不满，满足了也不会很满意
激励因素（成就/认可/成长）——满足了才能真正激励人。管理启示：涨工资只能消除不满，给成就才能激发动力

PDCA 循环与目标管理

工具内容
PDCAPlan（计划）→ Do（执行）→ Check（检查）→ Act（改进）。循环往复、螺旋上升。又称戴明环
目标管理德鲁克（Drucker）提出——上下级共同制定目标，以目标完成情况作为考核标准。核心：自我控制、参与管理

应用文练笔：写一份情况通报

题目情景

某市安全生产委员会办公室于 2026 年 5 月对全市工矿企业进行了安全生产专项检查，共检查企业 86 家。检查结果：大部分企业安全生产状况良好，但发现 3 家企业存在重大安全隐患（消防通道堵塞、危化品存储不规范、特种设备未年检），已责令停产整改。请以市安全生产委员会办公室名义向各县（市、区）安委会、各相关企业拟一份检查情况通报，既要通报问题提出批评，也要明确整改要求。

通报的格式要点

要素写法
标题XX关于XXX的通报。通报有三种：表彰通报/批评通报/情况通报。此题属于情况通报+批评结合
主送可多可泛——「各县（市、区）安委会，各相关企业：」
开头通报的背景（何时做了何事）
正文三段式——①总体情况（肯定好的）②存在问题（具体、点名或不点名地批评）③整改要求（限期+后续措施）
特点：先肯定后批评，批评具体化
结尾「特此通报」或自然收束。不写「妥否，请批复」（那不是通报的结尾）

⚠️ 通报和通知的区别再强调一次：通知是事前（让你去做什么），通报是事后（告诉你发生了什么+警示或教育）。题干如果是「检查完了，把结果通报给大家」→ 通报。昨天那个会议通知是「要开会了，通知大家来」→ 通知。

市安全生产委员会办公室
关于全市工矿企业安全生产专项检查情况的通报

各县（市、区）安委会，各相关企业：

为进一步压实安全生产责任，排查治理事故隐患，市安委会办公室于2026年5月15日至31日对全市86家工矿企业开展了安全生产专项检查。现将有关情况通报如下：

一、总体情况

从检查情况看，大多数企业能够落实安全生产主体责任，安全管理制度较为健全，隐患排查治理基本到位。其中，XX矿业有限公司、XX制造有限公司等12家企业安全生产标准化建设成效突出，予以通报表扬。

二、存在的主要问题

检查中发现3家企业存在重大安全隐患：XX化工有限公司危化品存储不符合规范要求，未落实双人双锁管理制度；XX建材有限公司生产车间消防通道严重堵塞，消防器材超过有效期未更换；XX机械有限公司特种设备超期未年检，部分起重设备安全装置失效。上述3家企业已当场责令停产整改。

三、下一步工作要求

（一）被通报批评的3家企业须于6月15日前完成全部隐患整改，经属地应急管理部门验收合格后书面报告市安委办，验收不合格的不得恢复生产。（二）各县（市、区）安委会对本次通报的问题要举一反三，于6月30日前对本辖区同类企业开展一次全覆盖排查，排查结果书面报送市安委办。（三）各企业要对照本次通报的问题清单全面开展自查自纠，切实消除安全盲区，确保安全生产形势持续稳定。

特此通报。

市安全生产委员会办公室
